# Student-Report-Management-System-using-Python
A Student Report Management System developed using Python. This project helps to manage student records efficiently, including adding, updating, deleting, and viewing student details and marks.

It is a simple console based application designed to demonstrate the use of file handling, functions, and basic Python concepts. This system makes it easy to store and manage student academic reports in an organized way.

🚀 Features

Add new student records
View student details
Update student information
Delete student records
Calculate total marks and percentage
File handling for data storage

🛠️ Technologies Used

Python,

File Handling,

Basic Data Structures,

🎯 Purpose

This project is created for learning and practicing Python programming concepts such as functions, loops, conditional statements, and file management.
